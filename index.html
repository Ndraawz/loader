Pemutar Video Sederhana

Informasi Penting:

Pastikan file youareanidiot.mp4 berada dalam folder yang sama dengan file HTML ini.

Jika video tidak muncul, periksa nama file dan pastikan sudah benar.

Penjelasan Kode:

Tag <video> digunakan untuk menampilkan video di halaman web.

Atribut controls menambahkan kontrol pemutaran (play, pause, volume, dll).

Tag <source> di dalamnya menentukan sumber video dan tipe filenya.
